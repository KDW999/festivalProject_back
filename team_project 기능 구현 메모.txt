기능 정리

1. localhost:3000을 치고 들어가면 메인 화면이 나옴

2. 메인 화면에서는 총 4개의 축제가 보여지고 4개 이상은 paging처리 및 한줄평은 첫번째 축제의 한줄평이 보여진다

2-1. 축제를 누르면 해당 축제에 관련된 한줄평이 보이고 해당 축제의 상세정보 및 후기 게시글을 볼 수 있다. 홈페이지를 누르면 해당 축제의 공식 홈페이지로 이동하게 되며 우측의 축제정보를 누르게 되면 해당 축제의 축제정보 글을 볼 수 있다.

2-2. 우측에서는 한줄평을 작성할 수 있다. 만약 로그인이 되어있지 않으면 로그인이 필요하다는 알림창을 띄우게 했다.

3. 만약 첫번째가 아닌 다른 축제를 클릭해서 되돌아가는 버튼이나 왼쪽 상단 festivalProject 버튼을 누르면 해당 축제의 한줄평이 메인화면에서 나온다.

4. 로그인 구현을 해놓았으며 비밀번호를 칠 당시 눈 아이콘을 한번 클릭하면 비밀번호를 볼 수 있으며 한번 더 누르면 비밀번호를 볼 수 없게 된다.

5. 회원가입 구현을 해놓았으며 8자리 이상, 체크버튼을 누르면 중복확인이 가능

6. 비밀번호, 비밀번호 확인은 영문자 + 숫자 + 특수문자(!?@_)를 포함한 8-20자
비밀번호가 같지 않다면 회원가입 불가

7. 닉네임도 8자리 이상이고 중복확인이 가능하며 옆의 avatar부분을 클릭하면 사진을 넣을 수 있다. 만약 중복된 닉네임이 있다면 이미 사용중인 닉네임이라고 뜬다. 한글 닉네임은 안되고 영문만 가능

8. 전화번호는 총 13자리로 ex) 010-0000-0000 형식으로 가능하다. 중복체크가 가능하며 만약 전화번호가 있다면 다음 페이지로 넘어갈 수 없다.

9. 다음 페이지를 누르면 회원가입하는 당사자가 관심있는 축제에 대해 체크를 할 수 있으며 하지 않아도 상관없다. 또한 체크를 하게 된다면 로그인 시 메인 화면에서 관심있는 축제가 하단에 뜨게 된다.

10. 축제 후기 게시판, 자유 게시판에 들어가면 다른 사람들이 작성해놓은 게시물을 볼 수 있고 추천을 누르거나 댓글을 작성할 수 있음.

11. 축제 후기 게시판에서 게시물 작성을 누르면 쓰고싶은 축제 이름을 검색할 수 있고 (무조건 축제를 넣어야한다.) 축제 이름을 클릭하면 옆에 축제 이름에 들어가게 된다. 가장 우측 이미지 버튼을 누르고 사진을 클릭하면 내용 칸에 사진이 들어가게 된다. 우측 연필 버튼을 누르면 게시된다.

11-1. 자신이 쓴 후기글이나 자유게시물은 수정 및 삭제가 가능하다.
11-2. 게시글에서 다음 글이나 이전 글을 누르면 다음 글과 이전 글로 이동하게 된다.
만약 다음 글이나 이전 글이 없다면 알림창이 뜨게 된다.

12. 자유 게시판 동일

13. 마이페이지로 들어가게 되면 자신이 쓴 후기 게시글이 나온다

14. 수정 버튼을 누르면 사진, 닉네임을 변경 할 수 있게 되고 완료를 누르게 된다면 둘 중 하나라도 바꿨을 시 변경되었다는 알림창이 뜬다.

15. 또한 회원가입 시 관심있는 축제를 체크 했으면 내 관심축제에 관심있는 축제들이 무엇인지 나열된다.

16. 로그아웃 버튼을 누르면 로그아웃이 되면서 메인 화면으로 이동하게 된다.
